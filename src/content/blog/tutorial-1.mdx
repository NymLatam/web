## 🔩 Qué es NymConnect?
NymConnect es una interfaz fácil de usar que le permite conectar otras aplicaciones a Nym mixnet para protección de la privacidad.

Use NymConnect para mejorar la privacidad Telegram, Electrum y Blockstream Green ( o cualquier aplicación que pueda ejecutar SOCKS5 ).

## 🔩 Cómo usar NymConnect?

## 🐧 NymConnect en sistema operativos linux

Primero vamos a la web de descargas de [Nymconnect](https://nymtech.net/download/nymconnect) 

El archivo que descargamos es una [`AppImage`](), este archivo al decsrgarse no tiene pernisos de ejecución por lo tanto debemos de darle estos permisos con el siguiente comando

```bash title="bash.sh" {2,3}
chmod +x nym-connect_1.1.21_amd64.AppImage
```

## 🔩 Seleccionar un Gateway 

Los Gateways Protege sirven como entrada a la mixnet. Verifican las credenciales de ancho de banda antes de enviar el tráfico a tres capas de los mixnodes. Los gateways serán recompensadas a través de una parte de las tarifas de los usuarios.

Por esto es importatnte escoger un Gate way que este geograficamente cerca de tu ubicación 

- Visita el [Explorador de Gateways](https://explorer.nymtech.net/network-components/gateways) para selecciona el más cercano a tí.
